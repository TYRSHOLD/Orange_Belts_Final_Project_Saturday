Sprint 9: Final Project – Productivity Dashboard Web App
Overview

In this final project, you will build a Personal Productivity Dashboard using HTML, CSS, and JavaScript. This project is designed to simulate a real-world web application that you can include in your portfolio and resume.

Your dashboard will allow users to manage tasks, track progress, and interact with a clean, responsive interface.

Project Goals
Apply all skills learned in the course
Build a polished, real-world web application
Practice writing clean, organized code
Prepare a project suitable for your portfolio
Required Features
1. Task Manager (CRUD Functionality)

Your app must allow users to:

Add new tasks
Mark tasks as complete
Delete tasks
(Optional but encouraged) Edit tasks
2. Local Storage

Your app must save data so it persists after refresh.

You must store:

Task list
Task completion status
3. Custom Fonts & UI Design
Use at least one Google Font or custom font
Design should look clean and professional
Focus on spacing, colors, and layout
4. CSS Animations

Include at least one animation:

Task appears with animation
Task completion effect (fade, strike-through, etc.)
Button hover effects
5. Responsive Design

Your app must work on:

Mobile devices
Tablets
Desktop screens

Use media queries or flexible layouts.

6. MP3 Sound Integration

Add sound effects using JavaScript:

Task added sound
Task completed sound
7. Dashboard Feature (Choose at least ONE)

Add a dashboard element such as:

Live clock
Daily quote generator
Progress bar (percentage of completed tasks)
Weather display (optional API)
Bonus Features (Optional)

Choose 1–2 for extra challenge:

Dark mode toggle (saved with local storage)
Task categories or tags
Drag-and-drop task reordering
Search or filter tasks
Due dates for tasks
Project Structure

Your project should include:

index.html
style.css
script.js
assets/ (for audio files, images, etc.)
Presentation Requirements

You will present your project to the class. Be ready to explain:

How your app works
How you used local storage
One challenge you faced and how you solved it
Submission Requirements
Completed working project
Clean and organized code
Responsive design
All required features implemented
Resume Example

You may use or adapt this description:

Built a responsive productivity dashboard web application using HTML, CSS, and JavaScript. Implemented task management with persistent data using local storage, interactive UI animations, and audio feedback for user actions.

Tips for Success
Start simple, then add features
Test your app often
Keep your code organized
Ask for help when needed
Final Note

This project is your opportunity to show everything you’ve learned. Make it something you are proud to share in your portfolio!
